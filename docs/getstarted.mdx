# Get started

## Installation
Install `de-code-product` or `de-code-marketing` from npm:

```bash
npm i de-code-product
```

## Usages
de-code is a CSS library something similar to twitter bootstrap and uikit.  de-code packages include a single CSS file which can be used by UI project just by importing the CSS file directly from the node_modules.

```js
import styles from 'node_modules/de-code/css/index.css';
```
### CDN
You can import de-code CSS directly from unpkg CDN
```js
<link src="https://unpkg.com/de-code-product/css/index.css" ref="stylesheet">
```
